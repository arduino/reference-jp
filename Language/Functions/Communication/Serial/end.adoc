[float]
=== 説明
シリアル通信を無効化し，RXとTXピンでの標準入出力を可能にします．シリアル通信を再開する場合には，link:../begin[Serial.begin()]を呼び出してください．
[%hardbreaks]


[float]
=== 構文
`Serial.end()`

_Arduino Megaのみ:_

`Serial1.end()` +
`Serial2.end()` +
`Serial3.end()` +


[float]
=== パラメータ
なし

[float]
=== 戻り値
なし
