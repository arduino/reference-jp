= bitClear()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== 説明
数値変数のビットをクリア（0に書き換え）します．
[%hardbreaks]


[float]
=== 構文
`bitClear(x, n)`


[float]
=== パラメータ
`x`: ビットをクリアしたい変数の名前

`n`: クリアしたいビットの指定，一番右である最下位ビット(LSB)が0

[float]
=== 戻り値
なし
