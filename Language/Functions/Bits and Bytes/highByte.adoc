= highByte()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== 説明
変数の高位バイト（左側）を取り出します．大きなデータ型の場合は，2番目に低いバイトを取り出します．
[%hardbreaks]


[float]
=== 構文
`highByte(x)`


[float]
=== パラメータ
`x`: 全てのデータ型の値

[float]
=== 戻り値
byte
